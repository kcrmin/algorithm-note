## 자료구조 선택
`array[]`: `id`가 숫자고 범위가 정직하면 
`hashMap`: 들어오는 `id` 수는 적은데 범위가 크면
`tree(Map/Set/...)`: 정렬된 상태로 관리
`Segment Tree`: 구간 연산 + 점 갱신 / 구간 갱신

## Time Version

class Soldier {
	int id;
	int score;
	int ver; // 2. heap 원소에도 version 저장
}


int[] version = new int[MAX]; // 1. 현재 상태 배열 준비


// 3. 최초 등록 시 version 올리고 넣기 (init)
version[id]++;
pq.add(new Soldier(id, score, version[id]));

// 4. Update도 삭제 말고 새로 넣기
version[id]++;
pq.add(new Soldier(id, newScore, version[id]++));

// 5. 조회할 때 TOP 검사
while (!pq.isEmpty()) {
	Soldier top = pq.peek();
	if (version[top.id] != top.ver) {
		pq.poll();
		continue;
	}
	return top.id;
}

// 6. Delete는 버전 갱신하고 아무것도 안넣기
alive[id] = false; // 방법 1
version[id]++; // 방법 2: 버전 올리고 새로 안넣기
