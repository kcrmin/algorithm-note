# B형 Practice Routine

> **한 줄 요약:** 짧은 주기로 반복하면서 `속도 + 정확도 + 복기`를 동시에 올리는 루틴.

## 주간 루틴 (권장)

- Day 1-2: 그래프 2문제 (`Dijkstra/MST/SCC`)
- Day 3: 구현 + 시뮬레이션 1문제
- Day 4: DFS/백트래킹 1문제
- Day 5: DP/게임 1문제
- Day 6: 실전 모의 2시간
- Day 7: 오답 정리 + 템플릿 보정

## 문제당 운영 시간

- 10분: 모델링 + 복잡도 예산
- 35분: 구현
- 10분: 디버깅
- 5분: 복기 메모

## 품질 기준

- 통과 후 1회 리팩터링: 변수/상태 명확화
